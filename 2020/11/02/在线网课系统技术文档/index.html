框架与工具

1.MyBatis-Plus

本项目主要使用的MyBatis-Plus功能：

  • 内置的Mapper, Service进行基本的CRUD操作
  • 主键的生成策略，默认全局唯一ID
  • 代码生成器
  • 内置的分页插件
  • 自动填充字段，比如创建时间和更新时间等，需要继承MetaObjectHandler接口
  • SQL性能分析
  • 条件构造器wrapper进行复杂的CRUD
  • 乐观锁插件
  • 逻辑删除插件
  • 。。。

2.MySQL

  • 设计参考《阿里巴巴Java开发手册》
  • 乐观锁机制，设置version标志位
  • 物理删除与逻辑删除，设置is_deleted标志位

3.SpringBoot

4.Swagger2

Swagger 是一个规范和完整的框架，用于生成、描述、调用和可视化 RESTful 风格的 Web 服务。

注意，在引入spring-security之后，需要配置不进行拦截的url

5.统一返回数据格式（自定义）

将所有的API的数据格式进行统一，封装成json返回，使前端对数据的操作更加一致。

6.log4j

7.ES6，ES5

Babel将ES6代码转为ES5

8.Vue

用于构建用户界面的渐进式框架，允许采用简洁的模板语法来声明式地将数据渲染进 DOM 的系统.

  • 基本的语法

  • 组件，可以扩展 HTML 元素，封装可重用的代码

  • 路由。路由允许我们通过不同的 URL 访问不同的内容，通过 Vue.js 可以实现多视图的单页Web应用（single page web application，SPA）

9.axios

axios 独立于vue 的一个项目，基于promise用于浏览器和node.js的http客户端

10.element-ui

element-ui 基于 Vue.js的后台组件库，方便进行页面快速布局和构建。基本上都是静态的方案，需要自己写动态的方案。

11.Node.js

Node.js 就是运行在服务端的 JavaScript

npm全称Node Package Manager，是node.js的包管理工具，相当于前端的maven
